# Amigo-Imaginario-AI-

## Create virtual environment
```bash
python -m venv venv
```

## Activate virtual environment
```bash
source venv/bin/activate
```

## Deactivate virtual environment
```bash
deactivate
```

## Install required libraries
```bash
pip install -r ./requirements.txt
```

## Run the project
```bash
python app.py
```
